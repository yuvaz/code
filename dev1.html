simple_list - This is a parent for the simple listing contain the generic bg color

simple_list_group - This class forms a group for a simple listing, group can contain any number of heading and item

simple_group_header - Heading for a group

simple_list_item - parent for every listing, contain the border, padding and cursor

simple_column - just a class to name the column

simple6040 - table structure left is 60% and right is 40%

simple_title_l1 - left block - Main title with overflow styles, in small screen the text exceed the defined width - it will be truncated with "..."

lnk_primary - generic class used to inject the branding color and size

simple_subtitle_l1 - right block - contain the ammount value in the right aligned column, it can contain multiple span for one line text

simple_title_l2 - 2nd row left block - contain the account type

simple_subtitle_l2 - 2nd row right block - contain the other currency type in the right aligned column, it can contain multiple span for one line text

adding to this two type of column structure, we can add any number of "p" to add new single row, "span" is used to inject dad inside that

This listing is mostly used in the sf-comp, so it will inject a active class for the selected listing, "selected" class to the parent of the "simple_list_item" - this will give a bg change and right green border for selected item

single_row - this class will give more spacing in top and bottom of the listing - it should be added in the same level of "simple_list_item" to create a proper click blocks
