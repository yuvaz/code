ACTIVER VOTRE SECURITY SMS

Le SMS Security renforce votre sécurité lorsque vous utilisé Easy banking.

  • Enregistrez ou choisissez le numéro de GSM belge sur lequel vous souhaitez recevoir le SMS security
  • Lorsqu'une opération le nécessite, vous recevrez, à ce numero, un SMS contenant un code
  • Introduisez ce code dans votre lecteur de carte
  • Introduisez ensuite la réponse obtenue sur votre lecteur de carte dans votre Easybanking

Le SMS security est parfetement sûr et gratuit et ne sera utilisé à aucune autre fin.
